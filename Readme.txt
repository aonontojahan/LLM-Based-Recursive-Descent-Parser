# LLM-Inspired Recursive Descent Parser

Course: CSC 461 – Programming Languages and Structures  
Semester: Fall 2025

## Overview
This project implements a recursive descent parser inspired by large language model parsing strategies. It supports operator precedence, unary operators, nested expressions, and graceful recovery from ambiguous or invalid input.

## Features
- Recursive descent parsing
- EBNF-based grammar
- LLM-inspired unknown token prediction
- Error handling and recovery
- User input support

## How to Run
```bash
python llm_recursive_descent_parser.py
